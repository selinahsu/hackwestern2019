Useful Resources 

SSH into the Pi
ssh pi@192.168.137.233 -v

Edit a file (in current directory): vim test.py
Once in vim, hit "I" to edit normally
Exit vim without saving: :qa!
Save and exit vim: :wq

Run a Python script (in current directory)
python test.py
